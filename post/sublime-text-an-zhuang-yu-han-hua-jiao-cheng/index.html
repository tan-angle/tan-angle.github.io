Sublime Text安装与汉化教程

2021-06-04 / 2 min read
1、官网下载

      https://www.sublimetext.com/

2、安装

      安装过程需要注意一下安装目录，默认是C盘，你可以更换一下安装目录，比如D、E、F盘等等（推荐）。当然如果你也可以不换，毕竟它的安装包还是比较小的。


安装成功后可以在电脑的开始菜单中打开。

3、配置国内插件源

      按“Ctrl+Shift+P”打开控制台，输入“pcac”后回车，之后会在当前页面出现一个搜索框，然后粘贴下面这个网址：http://packagecontrol.cn/channel_v3.json，最后按回车即可！

4、安装Package Control

很多人都是在这一步出了问题。
图片步骤：
① 按“Ctrl+Shift+P”打开控制台，输入“install”，在出现的下栏菜单选项中鼠标左键单击“Install Package Control”进行安装。

②鼠标左键点击“install package control”后可能会弹出这个页面，此时把它关闭，然后再把sublime text关掉再重新打开，最后按照同样的方法按“Ctrl+Shift+P”打开控制台，输入“install”，单击“Package Control”进行安装即可。

③ 点击“package control：install package”后，在当前页面最左下角会有一个“Loading”的一个滚动条（意思是正在加载中），等待一分钟左右即可！如果超过了一分钟还没有出现，把它关闭再重新打开即可！

④经过耐心等待后，会弹出一个搜索框：这就代表你已经成功安装了“package control：install package”。

5、安装中文语言包

      在弹出的搜索框中输入"Chinese”，点击下拉菜单栏中的“ChineseLocalizations"即可！

      之后，同样会在当面页面的左下角中加载安装包！

6、出现一下页面即为安装成功！

      剩下的内容自己探索吧！比如安装各种你所需要的插件（c语言、python等，还有代码自动补齐插件、主题美化等等）不懂的自己再查阅资料吧！
