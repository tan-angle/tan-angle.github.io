如何在Deepin操作系统V20版本中添加右键刷新按钮

2021-06-04 / 2 min read

一、一键自动安装

首先，需要在终端中安装wget：

sudo apt install wget

其次，再输入以下命令进行安装，安装过程中终端会提示：请选择使用F5刷新还是Fn+F5刷新。我一般选择F5。

sh -c "$(wget https://gitee.com/lyvnee/deepin-refresh/raw/master/install.sh -O -)"

二、手动安装

1.安装依赖

打开深度终端输入下面命令，安装Deepin文件管理上下文菜单 OEM 支持插件和键盘模拟软件xautomation。

sudo apt install dde-file-manager-menu-oem xautomation -y

2.创建desktop文件

在/usr/share/deepin/dde-file-manager/oem-menuextensions/路径下创建refresh.desktop文件，具体操作如下。

1）创建路径并新建refresh.desktop；

sudo mkdir -p /usr/share/deepin/dde-file-manager/oem-menuextensions/
sudo dedit  /usr/share/deepin/dde-file-manager/oem-menuextensions/refresh.desktop

2）填入以下内容后保存。

[Desktop Entry]
Type=Application
Name=refresh
Name[zh_CN]=刷新
Icon=view-refresh
Exec=xte "key F5"
#Exec=xte "keydown Multi_key" "key F5" "keyup Multi_key"
X-DFM-MenuTypes=EmptyArea

注意：请根据不同硬件条件需求来选择使用F5或Fn+F5进行刷新，然后调整上面#的位置。

所有步骤完成之后，需要退出文件管理器，然后重新打开。
